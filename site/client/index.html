Orthographe rationnelle

Les linguistes de l'association Érofa proposent trois règles qui rationalisent quelques points difficiles de l'orthographe du français.

Allez voir à quoi ressemble le résultat dans les textes suivant avant de lire ce que font les règles !

Notez que la transcription est automatique, et fait quelques fois des erreurs. Notez également que ces changements sont bâtis sur les rectifications de 1990, qui sont donc (en partie) appliquées.

Les règles

Vous avez dû constater que les textes se lisent bien, même sans explication, et vous avez peut-être deviné une partie des règles :

1.

Les consonnes doubles qui n'ont pas d'effet sur la prononciation sont simplifiées.
Exemples : sonnerie, appelle, accord, lutte, verre
Mais pas : messe, accident, ennui, surréalisme

2.

Les x finaux muets deviennent des s.
Exemples : bijoux, choix, veux, deux
Mais pas : dix, duplex, Chamonix

3.

Les ph, h et y grecs ou similaires sont simplifiés.
Exemples : photographie, rythme, humain, psychologie, chaos, théâtre, huile, déshabiller
Mais pas : hache, ahuri, pays, rugby, Hercule, hellénique

Outils de mise en pratique

Comme vous avez pu constater plus haut, nous proposons des outils pour utiliser cette proposition de rationalisation.

Notez que ces outils utilisent des dictionnaires construits à partir de sources dont la licence ne permet que les utilisations non-commerciales (Érofa et Lexique).

Transcription de documents

Les formats acceptés sont :

  • fichiers textes (.txt, .mkd, .md)
  • pages web (.html, .xhtml, .htmlz)
  • fichiers Word, LibreOffice, OpenOffice (.doc, .docx, .odt)
  • livres numériques (.epub), comme ceux de wikisource plus haut
  • fichiers PDF : pas directement supporté
    Si vous avez le fichier original utilisé pour créer le PDF, transcrivez ça, le résultat sera nettement meilleur.
    Si vous n'avez pas le fichier original, vous pouvez transformer le fichier PDF en fichier Word (par exemple avec Adobe ou Pdf24), puis soumettre le fichier Word ici.

Transcription de documents sur votre ordinateur

Téléchargez le programme de transcription.

Notez que c'est un programme pour Linux, et sans interface graphique, seulement à la ligne de commande. Pour Windows, il est possible que cela marche sous WSL. Il est peut-être possible de compiler ce programme pour MacOS et Windows, à partir du dépôt source indiqué plus bas.

Transcription des pages internet que vous visitez

  • pour Firefox

  • pour Microsoft Edge, suivez ces instructions (deux premiers points), puis installez l'extension ici

Vérificateurs d'orthographe

Afin de pouvoir écrire en orthographe Érofa sans que votre ordinateur ne veuille vous corriger en permanence (en soulignant en rouge), vous pouvez enseigner l'orthographe Érofa :

  • à Firefox (navigateur internet, vérification non supportée sur téléphones)
  • à Thunderbird (app d'emails/agenda/contacts)
  • à OpenOffice (suite bureautique)

Support pour d'autres orthographes...

Parmi nos outils, voici ceux qui supportent d'autres orthographes que l'orthographe Érofa :

  • transcription interactive
  • transcription de documents (voir plus bas)
  • transcription de documents sur votre ordinateur (voir --help)
  • transcription de pages que vous visitez (voir les options de l'extension)
  • vérificateurs d'orthographe

Notez que ces autres orthographes ne sont pas des recommendations (ni des futures recommendations), simplement des possibilités d'expérimentation.

Données

Nous fournissons des données brutes qui décrivent le passage de l'orthographe usuelle à d'autres orthographes :
