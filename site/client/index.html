Ortografe simplifiée

Les linguistes de l'association Érofa proposent trois règles pour simplifier des points difficiles de l'orthographe du français sans choquer :

1.

Les consonnes doubles qui n'ont pas d'effet sur la prononciation sont simplifiées.
Exemples : sonnerie, appelle, accord, lutte, verre
Mais pas : messe, accident, ennui, surréalisme

2.

Les x finaux muets deviennent des s.
Exemples : bijoux, choix, veux, deux
Mais pas : dix, duplex, Chamonix

3.

Les ph, h et y grecs ou similaires sont simplifiés.
Exemples : photographie, rythme, humain, psychologie, chaos, théâtre, huile, déshabiller
Mais pas : hache, ahuri, pays, rugby, Hercule, hellénique

Transcription de documents

Les formats acceptés sont :

  • fichiers textes (.txt, .mkd, .md)
  • pages web (.html, .xhtml, .htmlz)
  • fichiers Word, LibreOffice, OpenOffice (.doc, .docx)
  • livres numériques (.epub)
    Vous pouvez trouver sur internet des livres classiques au format numérique, par exemple Capitaine Fracasse, Les trois mousquetaires ou Vingt mille lieues sous les mers sur wikisource.
  • fichiers PDF : pas directement supporté
    Si vous avez le fichier original utilisé pour créer le PDF, convertissez ça, le résultat sera nettement meilleur.
    Si vous n'avez pas le fichier original, vous pouvez transformer le fichier PDF en fichier Word (par exemple avec Adobe ou Pdf24), puis soumettre le fichier Word ici.
